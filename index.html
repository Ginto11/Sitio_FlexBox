Pagina con FlexBox

Esta pagina fue realizada, con el fin de poner a prueba el conocimiento acerca de flexbox el cual sirve para el posicionamiento de los elementos de Dom y por consiguiente aqui explico el funcionamiento de sus atributos.
Para trabajar con FlexBox se tienen que tener en cuenta varias cosas o acciones. Como que el contenedor padre debe tener un display flex para utilizar todos los atributos correspondientes

Atributos

  • flex-direction: Sirve para estipular como van a estar organizados los elementos si en columnas o filas y su valores son:
    • row
    • column
    • row-reverse
    • column-reverse
  • flex-wrap: Esto permite que cada elemento que supere el tamaño del contenedor padre se vaya poniendo debajo sucesivamente y sus valores son:
    • wrap
    • nowrap
    • wrap-reverse
  • align-items: Sirve para posicionar elementos verticamente y sus valores son:
    • center
    • space-around
    • flex-start
    • flex-end
    • stretch
    • baseline
  • justify-content: Se utiliza cuando se quiere posicionar elementos horizontalmente y sus valores son:
    • center
    • space-around
    • space-between
    • flex-end
    • flex-start
  • align-content: Sirve para agrupar elementos que esten en wrap y sus valores son:
    • center
    • flex-end
    • flex-start
    • space-around
    • space-between
